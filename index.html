Tango4 - another implementation of distributed offline storage

Tango4 is another implementation of distributed offline storage for web-applications, with synchronisation server built with JavaScript on client and server, backed by node.js and PostgreSQL on server and IngexedDB on client side

Lima1 related projects:

Projects listed here share same idea and service of offline object data storage for Web and Mobile

  • Lima1 holds general idea of storage library and implementation for Android
  • Lima1OpenShift is the reference implementation of sync service runs on Redhat OpenShift cloud platform

There are also two client reference projects, both have Web and Android applications:

  • StickStack is Adobe Air/Phonegap powered sticky notes taking application, utilizes first revision of Lima1 protocol
  • Whiskey2 uses latest protocol and Android sync service. Has native Android application and Chrome Web application

Android plugins related projects:

Projects grouped here are heavily use AIDL technology

  • Sierra5 pair of Android applications. One is simple plain text files browser and editor, another is collection of plugins and services extending functionality of first one
  • Quebec4 is quick media collector application. Collected data is provided to clients via AIDL. Sierra5 has a consuming plugin for it
  • Foxtrot7 is slightly different idea, but shares same technology. Focus here is moved to communication over Bluetooth for Desktop and Android

Other projects:

  • XRay2 handy Desktop utility for Japanese language learners build with Adobe AIR and jQuery Mobile
  • mobileorg-android the only Android application published in Play Store. EMacs Org mode client, not actively maintained now.
